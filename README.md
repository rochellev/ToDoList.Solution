# To To List : The C# Web App
### By Rochelle Roberts
-----

## Description
* To Do List C# Web App

## Technologies Used
* C#
* .NET
* ASP.NET Core MVC
* HTML
* MSTest

## Installation
* Clone project repo
* From the project's root directory, open console and type dotnet run

```sh
$ git clone 
$ cd  
$ dotnet run
```
