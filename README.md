## Installation
* Follow steps below
* Go to http://localhost:5000


```sh
$ git clone https://github.com/rochellev/ToDoList.Solution.git
$ cd  .\ToDoList.Solution\ToDoList
$ dotnet restore
$ dotnet run
```
